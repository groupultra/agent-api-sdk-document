> 有如下类型的 payload

- User 发起的

  - User Login
    - user_login
  - User Event
    - message_up
    - button_click
  - Other User Actions
    - fetch
    - roger
    - heartbeat

- Service 发起的
  - Service Login
    - service_login
  - Service Event
    - message_down
    - update
  - Other Service Actions
    - heartbeat
- Moobius 发起的
  - copy
